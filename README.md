# Quant Trading Data - GitHub Pages
This repository hosts JSON data for Birdie Quest dashboard.

## Files:
- trades.json: Closed trades with reason codes
- signals.json: Current signals with date/time/strength/model
- sessions.json: Session performance with dates
- positions.json: Open positions with model attribution
- health.json: System health status

Updated every 5 minutes from local Enhanced V2 API.
